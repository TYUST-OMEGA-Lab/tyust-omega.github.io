# 🌌 欢迎来到 TYUST OMEGA Lab

**OMEGA** 代表 **O**ptimal / **O**pen-world / **O**mni Machine lEarning on **G**raph d**A**ta。

我们是一个专注于**图数据挖掘技术**、**图神经网络 (GNN)** 及**图自监督学习**的前沿研究小组。我们的研究致力于打通从底层优化理论到开放环境适配，再到跨学科广泛应用的完整链路。

### 核心理念：Ω³ / Ωⁿ
实验室的研究由三个互补的维度构成（$\Omega^3$），并追求如幂函数般的无界增长（$\Omega^n$）：

* **Optimal (优化理论)**：由博士青年老师领衔，主攻 GNN 的理论边界、表达力提升及算法结构上的本质突破，为实验室提供坚实的底层性能支撑。
* **Open-world (开放环境)**：针对现实世界的复杂性，研究图对比学习、开放图半监督、零样本学习及图数据稠化/补全技术，使模型具备走出实验室、应对未知环境的能力。
* **Omni (广泛应用)**：将先进算法深度应用于 **点云处理 (3D Vision)**、**蛋白质与分子科学 (AI for Science)**、**天文大数据挖掘** 及人体姿态识别等跨学科领域。

> **寓意**：“OMEGA” ($\Omega$) 象征着学术探索的“圆满与成就”；而“幂” ($\Omega^n$) 则呼应了我们不断挑战现有理论瓶颈、突破科研边界的实验室精神。
